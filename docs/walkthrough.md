# 애플리케이션 동작 시연 (Walkthrough)

요청하신 대로 직접 브라우저를 실행하여 애플리케이션의 주요 기능을 시연했습니다.

## 1. 동작 시연 영상

아래 영상에서 다음 동작을 확인하실 수 있습니다:
1.  **메인 페이지 탐색**: Hero 섹션부터 하단 Contact 섹션까지 스크롤 탐색
2.  **블로그 기능**: [Curl Blog Post] 글 상세 조회 및 목록 복귀
3.  **프로젝트 기능**: [AI Analytics Dashboard] 프로젝트 상세 조회 및 목록 복귀
4.  **UI/UX**: 섹션 간 이동 및 반응형 디자인 확인

## 2. 검증 결과
-   **서버 연결**: 프론트엔드(`:3000`)와 백엔드(`:4000`)가 정상적으로 연동되어 데이터를 주고받습니다.
-   **데이터 로딩**: 프로젝트 및 블로그 데이터가 API를 통해 정상적으로 로드됩니다.
-   **이동성**: 각 섹션 링크 및 뒤로가기 동작이 매끄럽게 작동합니다.
